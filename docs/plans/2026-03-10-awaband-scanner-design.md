# AWABAND Scanner — Design Document

## Концепция

Веб-приложение, имитирующее работу браслета AWABAND из мира Awaterra 2225. Камера и микрофон телефона анализируют реальные физиологические показатели, которые транслируются в 7 параметров биополя. Пользователь видит свою «ауру» в реальном времени.

## Целевая аудитория

Фанаты вселенной Awaterra — промо и погружение в мир.

## Платформа

Веб-приложение (PWA-ready). Открывается в браузере на телефоне по ссылке.

## Технологии под капотом

### Камера (rPPG)
- Алгоритм CHROM/POS для извлечения пульсовой волны из видео лица
- Измеряемые параметры: HR (пульс), HRV (RMSSD), частота дыхания, Heart Coherence
- Референсы: pyVHR, Binah.ai, MTTS-CAN

### Микрофон (Voice Biomarkers)
- Web Audio API + AnalyserNode
- Измеряемые параметры: F0 (pitch), jitter, shimmer, HNR, RMS (громкость), spectral centroid
- Референсы: Kintsugi Health, Sonde Health

## Маппинг данных → 7 параметров биополя

| # | Параметр | Цвет | Частота | Источник данных |
|---|----------|------|---------|----------------|
| 1 | Стабильность | Красный | 396 Hz | HRV (RMSSD) — низкая вариабельность = высокая стабильность |
| 2 | Поток | Оранжевый | 417 Hz | Плавность пульсовой волны (отсутствие резких скачков HR) |
| 3 | Энергия | Жёлтый | 528 Hz | HR относительно нормы + громкость голоса (RMS) |
| 4 | Резонанс | Зелёный | 639 Hz | Heart Coherence (спектральный пик HRV ~0.1 Hz) |
| 5 | Вибрация | Голубой | 741 Hz | Pitch (F0) + богатство обертонов (spectral centroid) |
| 6 | Ясность | Индиго | 852 Hz | Чистота голоса (HNR) + стабильность pitch (низкий jitter) |
| 7 | Целостность | Фиолетовый | 963 Hz | Средневзвешенная согласованность параметров 1-6 |

Каждый параметр — значение 0-100. Светимость — взвешенное среднее всех семи.

## Экраны

### 1. Заставка
Стилизация под интерфейс AWABAND. «Поднесите устройство к лицу для сканирования».

### 2. Сканирование (основной)
- Видео с камеры (полупрозрачное) в центре
- Силуэт/контур поверх видео
- 7 слоёв цветного свечения (аура) вокруг силуэта — интенсивность = значение параметра
- Пульсация ауры синхронизирована с реальным пульсом
- Панель 7 полосок AWABAND снизу, обновляется в реальном времени
- Светимость (общий тон) сверху
- Свободный формат — камера и микрофон работают пока пользователь не нажмёт «стоп»

### 3. Результат
- Фиксированный снимок биополя
- Значения 7 параметров
- Возможность сохранить/поделиться

## Стек

- **Frontend:** Vanilla TypeScript, Canvas/WebGL
- **rPPG:** CHROM-алгоритм портированный на JS (без ML-зависимостей)
- **Аудио:** Web Audio API + AnalyserNode
- **Деплой:** Railway (существующий проект) или отдельный домен

## Архитектура

Полностью клиентское SPA. Вся обработка в браузере, бэкенд не нужен (кроме статик-сервера).

```
index.html
src/
  main.ts          — точка входа, роутинг экранов
  camera/
    rppg.ts        — CHROM-алгоритм, извлечение пульсовой волны
    face-detect.ts — детекция лица для ROI
    vitals.ts      — HR, HRV, breathing rate, coherence из rPPG-сигнала
  audio/
    voice.ts       — pitch, jitter, shimmer, HNR, RMS, spectral centroid
  biofield/
    mapper.ts      — маппинг сырых данных → 7 параметров (0-100)
    luminosity.ts  — расчёт Светимости
  ui/
    splash.ts      — экран заставки
    scanner.ts     — основной экран сканирования
    result.ts      — экран результата
    aura.ts        — WebGL-рендер ауры (7 слоёв свечения)
    awaband.ts     — панель 7 полосок AWABAND
```
